Drone Pursuit Tracking Project (SkyPro)

AI-powered drone tracking system using Python and OpenCV.

Overview

SkyPro is a cutting-edge AI-integrated drone system that detects and pursues moving targets in real time. Built in Python with OpenCV, it leverages simulated Lidar data for enhanced accuracy and responsiveness. Initially conceptualized for law enforcement to reduce risks in high-speed pursuits, the system demonstrates how Generative AI and hardware can converge to deliver transformative capabilities.

My Role & Key Contributions

  • Computer Vision Integration: Developed and optimized the Python scripts for real-time object detection using OpenCV, ensuring minimal latency.
  • Data Fusion & Simulated Lidar: Integrated the Lidar simulation into the drone’s tracking logic, enhancing accuracy and robustness.
  • Performance Optimization: Improved code efficiency (e.g., image processing pipeline) for smoother tracking and lower CPU usage.
  • Team Collaboration & Version Control: Used GitHub to manage merges, reviews, and issue tracking, ensuring a clean workflow for all team members.

Challenges & Outcomes

  • Challenge: Achieving low-latency video processing on limited hardware.
    Solution: Optimized Python scripts (OpenCV) by reducing overhead, using lightweight models like YOLOv3-Tiny, and minimizing unneeded data transfers.
  • Challenge: Ensuring stable landing on a moving target (magnet lock).
    Solution: Tuned the drone’s flight controller parameters in a simulated environment; implemented safety checks for consistent hover and lock engagement.
  • Outcome: Created a functional prototype demonstrating autonomous pursuit, reduced high-speed pursuit risks, and showcased real-time AI-driven decision-making.
  • Takeaway: Gained deep experience in computer vision, drone navigation, and multi-threaded programming for real-time systems.

Key Features

  • Real-time target detection and tracking using OpenCV.
  • Simulated Lidar data integration for enhanced accuracy.
  • Optimized Python code for efficient performance.
  • Modular design for easy scalability and maintenance.
